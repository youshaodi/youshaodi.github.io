水浒传

白胜 ( 象形武丑脸 ) ，《生辰纲》一据中的人物，在黄泥岗乔妆卖酒人。

晁盖 ( 黄三块瓦老睑 ) ，《浔阳楼》一剧中的人物。晁盖为梁山首领，率领众英雄救出宋江。

高俅 ( 水白整脸 ) ，《野猪林》一剧中的人物。高俅是宋太尉，助子霸占十万禁军教头林冲之妻，并诬林冲行刺，将林问罪刺配沧州。
公孙肚 ( 紫道士脸 ) ，《黄泥岗》一剧中的主角之一。讲粱山好汉晁盖与公孙胜、吴用、刘唐、白胜、阮氏三兄弟等人，在黄泥岗用计诱使杨志和脚夫饮蒙汗药洒后，劫职生辰纲的故事。《黄泥岗》、《浔阳楼》、《野猪林》、《清风山》、《收关胜》、《连环马》、《青州府》、《大名府》、《花田错》、《雁翎甲》、《石秀探庄》、《三打祝家庄》、《闹江州》等剧，均据《水浒传》改编而成。《水浒传》，也叫《水浒全传》，元末明初入施耐庵所作长篇小说。全书以描写农民战争为主要题材，塑造了李逵、武松、林冲、鲁智深等人物性格鲜明的梁山英雄人物，揭示了当时的社会矛盾。

关胜 ( 红三块瓦脸 ) ，《收关胜》一剧中的的主角。关胜为官府攻梁山，呼延灼诈降，诱关胜劫寨，被梁山头领合力擒获，劝关聚义。

呼延灼 ( 黑碎花脸 ) ，《连环马》一剧中的人物。故事讲呼延灼领兵攻梁山，摆布连环马，梁山众好汉不敌，派时迁盗回雁翎甲，并骗徐宁上山，操练钩连枪法，大破连环马。 151 ．秦明 ( 红碎花脸 ) ，《青州府》一剧中的人物，原是青州府的守将，讨伐清风山时为花荣计擒，被迫上山聚义。
金眼僧 ( 紫僧脸 ) ，《二龙山》一剧中的人物，出自《水浒传》故事。史文恭邀金眼僧、银眼僧、小和尚助战，三僧宿二龙山张青、孙二娘店中。张、孙夜刺三僧，二僧亡，小僧逃走。
李逵 ( 黑碎花睑 ) ，《黑旋风》一剧中的主角。故事讲丁甲山山寇冒梁山首领宋江之名强抢民女，李逵不问真情，怒砍杏黄旗，责问宋江。后李知错负荆向宋请罪。此剧出自《后水浒》。

刘唐 ( 蓝十字门葫芦脸 ) ，《浔阳楼》一剧中的人物。宋江浔阳楼醉题“反诗”被告密：在被出斩时，粱山好汉刘唐等劫法场，救出宋江。

鲁智深 ( 白僧睑 ) ，《野猪林》一剧中的主角之一。北宋禁军教头林冲遭陷，发配沧州，途经野猪林时，解差奉命欲在途中加害，鲁智深跟踪保护，林方幸免于难。
