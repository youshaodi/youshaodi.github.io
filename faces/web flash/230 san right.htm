三国演义

曹操 ( 水白整脸 ) ，《群英会》一剧中主角之一。建安十三年 ( 公元 208 年 ) ，曹操 (155 ― 220) 率军南下，欲扫灭孙权、刘备。后中庞统连环计，在江中大宴文武，横槊赋诗，此时黄盖诈降，火烧战船，曹军大败。

蔡阳 ( 白尖老三块瓦睑 ) ，《古城会》一剧中的人物。关羽辞曹寻兄，途中闻张飞已据古城，奉两嫂命往见。张飞以关羽在曹营日久，疑其有诈而责诘之。关羽正辩白间，曹将蔡阳率军适至，张飞益生疑，关羽立斩蔡阳，兄弟方释疑。
程普 ( 紫三块瓦脸 ) ，《凤凰二乔》一剧中的人物。孙策 (175 ― 200) 率周瑜、程普、黄盖投袁术借兵，凤凰寨乔玄将两位女儿许配孙策和周瑜，孙、乔合兵后大破严白虎。
周仓 ( 瓦灰花元宝脸 ) ，《华容道》一剧中的人物。周仓是关羽的亲随猛将。华容道中大喝一声，将曹操吓得惊慌失措。

邓艾 ( 白尖三块瓦脸 ) ，《坛山谷》一剧中的人物。姜维伐魏，魏大将邓艾 (197 ― 264) 领兵拒之。姜、邓斗智，姜在坛山谷设伏，故弃粮草，诱邓劫粮，邓中计大败。

典韦 ( 黄花三块瓦脸 ) ，《战宛城》一剧中的人物，曹操的猛将，因双戟被盗，在曹营被夜袭时战死。

董卓 ( 水白整睑 ) ，《凤仪亭》―剧中的人物。东汉献帝的太师，因专权，司徒王允授连环计，离间董卓、吕布父子。先将义女貂蝉许配吕布，又献给董卓，导致凤仪亭前父子反目，后来，董被吕刺杀。

关羽 ( 红整脸 ) ，《华容道》一剧中的主角，京剧《赤壁之战》中的一折。赤壁之战，曹操大败，诸葛亮遣将分路堵击曹操，独不遣关羽 (? 一 219) 。关羽力争出战，诸葛亮令其在华容道设伏。曹操率败军十八骑经华容道，哀恳关羽放行，关羽念曹操昔日相待之情，放其逃行。

郭淮 ( 黑碎花睑脸 ) ，《铁笼山》一剧中的人物。姜维在铁笼山败走，魏大将郭淮紧迫不舍，姜维手中只剩弓一张，郭淮用箭射姜，姜用手接箭回射，反使郭淮，中箭身亡。

黄盖 ( 红六分脸 ) ，《群英会》一剧中的人物。他与周瑜定下苦肉计，在群英会上周怒责黄盖，以便黄诈降曹操，火烧战船。在赤壁大战中黄盖立下奇功。

贾华 ( 倒元宝丑脸 ) ，《甘露寺》一剧中的人物，吴主孙权的大将，奉命埋伏甘露寺欲害刘备，计泄被国太所责。

蒋干 ( 文丑脸 ) ，《群英会》一剧中的人物，曹操的谋士和说客，因与吴国都督周瑜有同窗之谊，奉曹操命前去探听，拟劝降周瑜，却反中周的反间计，将假书信盗回曹营。曹操中计，乐了水军都督蔡瑁、张允。

蒋钦 ( 蓝尖三块瓦脸 ) ，《甘露寺》一剧中的人物，周瑜的大将。在刘备招亲后回荆州的路上，因追截刘备，被郡主孙尚香斥退。

姜维 ( 红三块瓦脸 ) ，《铁笼山》一剧中的主角。姜维是诸葛亮的弟子，他兴兵伐魏，困司马师于铁笼山，魏将陈泰搬来救兵，打败姜维。姜箭射魏将郭淮后，急退汉中。

孔秀 ( 紫尖三块瓦脸 ) ，《过五关》一剧中的人物，曹军东岭关守将，关羽过五关时孔不肯放行，被关羽斩首。
凌统 ( 紫三块瓦脸 ) ，《百骑劫魏营》一剧中的人物，吴国大将。他出战不利，被甘宁所救。甘宁又率百骑劫魏营，曹操大骇，遂与孙权议和。
吕蒙 ( 蓝尖三块瓦脸 ) ，《走麦城》一剧中的人物，东吴的都督，他乘关羽攻樊城之机，率军白衣渡江，偷袭荆州，使关羽中伏遇害。

作者：眉间的红格线条暗示他的命运。

马谡 ( 油白三块瓦脸 ) ，《失街亭》一剧中的人物，诸葛亮帐下的参军。诸葛亮命其镇守街亭，因刚愎自用，不听副将王平劝阻，擅自在山顶扎营，被围后失守街亭，为整军纪，诸葛亮挥泪将他斩首。

孟达 ( 倒元宝脸 ) ，《走麦城》一剧中的人物。《走麦城》演的是关羽骄兵轻敌，被曹操和孙权联合，前后夹攻，兵败走麦城的故事。关羽被困麦城向蜀将孟达求援，孟坐视不救，使关羽孤军突围遇害，孟达降曹。
孟谭 ( 黑碎花脸 ) ，《过五关》一剧中的人物。《挑袍》、《过五关》、《古城会》是三本塑造关羽神威凛然舞台形象的戏，统称之为《千里走单骑》。《过五关》一戏讲关羽虽受曹操厚礼相待，但仍思念着刘备，终于辞曹寻刘。孟谭是洛阳守将韩福的牙将，关羽过五关时，因阻拦被刀劈马下。

庞统 ( 紫道士脸 ) ，《来阳县》一剧中的人物。庞投奔刘备时，刘只授其来阳县令，庞到任后不理政事，张飞巡视责之，庞用半日剖判百日之事，使张飞敬服，遂告刘备授与副军师之职。《来阳县》、《华容道》、《战长沙》、《过巴州》、《走麦城》、《连营寨》、《铁笼山》、《小桃园》、《定军山》、《坛山谷》、《群英会》、《博望坡》、《战宛城》、《长坂坡》、《过五关》、《古城会》、《空城计》、《凤仪亭》、《甘露寺》、《失街亭》、《凤凰二乔》、《百骑劫魏营》等剧目，均取材于中国古典长篇小说．《三国演义》。

沙摩柯 ( 红碎花脸 ) ，《连营寨》一剧中的人物，西羌大将。
司马师 ( 红十字门花脸 ) ，《铁笼山》一剧中的人物，魏国大将，司马懿的儿子，被姜维围于铁笼山，缺水难忍，拜泉得水。魏将陈泰来救，司马师方得脱险。

司马懿 ( 水白整脸 ) ，《空城计》一剧中的人物。《空城计》演的是诸葛亮智退魏兵的故事。公元 228 年，蜀魏两国为争夺街亭，因蜀国大将马谡擅自改变诸葛亮的部署，被魏将杀得丢监弃甲。司马懿夺得街亭后，率大军直逼西城。诸葛亮因街亭失守，西城空虚，无兵将可遣，只得大开城门，故作镇静，在城楼抚琴，施空城计，以引起司马懿的疑心。司马懿果然中计，不取空城，反而退兵。

孙权 ( 水白整睑 ) ，《甘露寺》一剧中的人物，吴国大帝，因刘备占据荆州不还，与周瑜设下美人计，假称以妹孙尚香妻刘，拟诓刘过江，拘为人质，以索取荆州。刘用诸葛亮的计策，请东吴老臣乔玄劝说孙权之母国太许婚。孙母至甘露寺相亲，看中刘备，招赘为婿，使孙、周的美人计弄假成真。

张苞 ( 黑十字门花脸 ) 《小桃园》一剧中的人物。刘备为关羽、张飞复仇，兴兵伐吴，关羽子关兴，张飞子张苞夺印相争，刘备训以父辈桃园之义，二人结为兄弟，共报父仇

张飞 ( 黑十字门蝴蝶睑 ) ，《芦花荡》一剧中的主角，取材于《三国演义》，京剧《甘露寺》中的一折。《三国演义》是中国著名古典长篇历史小说，元末明初罗贯中 ( 约 1330 一 1400 之间 ) 根据陈寿《三国志》、范晔《后汉书》、元代的《三国志平话》和某些有关传说，经综合熔裁再创作而成。它描写了公元 184 年到 280 年间东汉末年和魏、蜀、吴三国的历史故事，集中地描绘了这一时期各封建统治集团之间的矛盾和斗争，塑造了刘备、关羽、张飞、诸葛亮、曹操、孙权等许多历史人物形象。对战争的描写注意分析矛盾的特殊性，用不同的战术，解决不同的战役，因而在他笔下描写的大大小小无数次战斗，都能干变万化，各有特色。“三国戏”就是根据这些生动故事改变的。张飞是蜀汉大将，在奉丞相诸葛亮之命，假扮渔夫埋伏芦花荡，接应刘备回荆州时，擒住周瑜又纵之，使周瑜气极，呕血而亡。
