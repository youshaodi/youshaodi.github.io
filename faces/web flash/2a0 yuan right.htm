元明清

蔡天化 ( 白花三块瓦脸 ) ，《淮安府》一剧中的大盗，他至淮安府盗去施世纶印信，匿于北极观，黄天霸等四处寻访，黄的义侄贺人杰将印信盗回，并合力擒住蔡天化。
窦尔敦 ( 蓝花三块瓦脸 ) ，《盗御马》一剧中的主角，此剧取材于《施公案》。窦尔敦是连环套寨主，与镖客黄三太因有宿仇，借盗太尉梁九公由皇帝所赐御马，图陷黄于罪。黄三太已死，官府令黄子天霸访拿盗马人，黄在好友朱光祖的帮助―下，查清盗马人，以情感使窦献出御马，随黄投案。
窦虎 ( 黄歪花脸 ) ，是《虮蜡庙》一剧中土豪费德功游虮蜡庙时，强抢民女的打手。
郝文僧 ( 白僧睑 ) ，《东昌府》一剧中的人物，此剧出自《施公案》故事。
郝摇旗 ( 揉红整脸 ) ，《闯王旗》一剧中李自成起义军将领之一。
贺天彪 ( 绿碎花脸 ) ，《连环套》一剧中的人物，在口外连环套与窦义敦聚义，为窦的部下。
蒋忠 ( 黑花元宝脸 ) ，《百凉楼》一剧中的人物。此剧取材于《英烈传》。蒋忠是朱元璋的部将，在百凉楼保卫朱元璋时立功。
郎如豹 ( 瓦灰老三块瓦脸 ) ，《里海坞》一剧中的山东土豪，在逼婚不成杀韩翠云后，又盗施世纶的印信，被黄天霸设计将印盗回，并将郎如豹拿住法办。
书僮 ( 小丑脸 ) ，《盗银壶》一剧中的人物，取材于《二刻拍案惊奇》。《二刻》为明人凌漾初编著，是作者根据他所闻的古今故事素材，以话本形式来创作的作品。
苏三 ( 旦 ) ，《女起解》一剧中的主角之一，此剧取材于：《警世通言》。崇公道，是老解差，心地善良，在提解苏三 ( 妓名玉堂春 ) 至太原复审途中，听说苏三遭遇，崇加以劝慰，并收为义女。
谢虎 ( 蓝花三块瓦脸 ) ，《―支桃》一剧中的人物。谢是淫贼，绰号“一枝桃”，常采花杀人，黄天霸奉命缉拿，在|州庙中相遇格斗，黄中计被谢用毒镖打伤，朱光祖赶至救黄，谢虎逃走。
