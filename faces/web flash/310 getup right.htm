传统脸谱: 脸谱文化的繁荣

 

一 平民

京剧等中国传统戏剧大多是从民间发展起来的民间文化。京剧的早期观众，大都是大字不识一个的文盲。这些观众文化水平很低、历史文学知识极其有限。面对这样的观众，怎 么才能让他们明白戏剧里讲的是什么呢？现在常用的每人发一份剧情简介的方法当然不行，不识字的人根本看不懂。中国早期的编剧家们为了解决这个难题，想出了 “ 脸谱 ” 这个妙法。由于当时人们的精神生活的匮乏，只将戏剧人物划分成好人、坏人。画红脸的就是忠臣，画白脸的就是奸臣，让三岁的小儿也一看就明白。

仅仅将人物划分成红脸和白脸，还是不易区分戏剧中众多的人物。于是，人们将它更加的具体化，加入了更加丰富的色彩，更加多变的图案。而且为了区分角色，在每个人物的脸谱中都加入一些用以区别的特征，使每个人物的脸谱各不相同。久之，这些特征成为了人们的共识，也就形成了有固定格式的脸谱。

其实正是这种脸谱化的人物、公式化的剧情，才使乡村的低文化观众喜欢看戏，也才使京剧等传统戏剧迅速在中国的老百姓中间普及开来，成为当时中国庶民的主要娱乐方式。

二 清朝宫廷

清乾隆五十五年（ 1790 年），以高朗亭为首的第一个徽班（三庆班）进入北京，参加乾隆帝八十寿辰庆祝演出。随后还有不少徽班陆续进京，著名的为四喜、春台、和春三班。从此，京剧进入了宫廷。“四大徽班”进京给京城观众以耳目一新之感，以其通俗质朴之气赢得了京城观众的欢迎，从此在京城扎下了根。

同治帝、光绪帝和慈禧太后都嗜好京剧。他们一方面组织“内学” ( 由太监们组成 ) 在宫中粉墨登场，另一方面还召“外学” ( 由民间艺人组成 ) 到宫内演出所谓“伺侯戏”，当时著名的演职人员无一例外地被编入“外学”中。不仅如此，同治、光绪二帝均能演戏，慈禧还亲自组织臣僚编戏。

 

北京・颐和园・德和园・大戏楼


德和园大戏楼是中国现存最大的古戏楼，建筑得很有特色，翘角重檐，朱栏绿柱，由专供慈禧看戏的颐乐殿及其两庑供王公大臣看戏的看戏廊组成。它高21米，分上下三层，三层舞台之间有天井相通。下层天花板中心有天井与上层戏台串通，中层戏台设有绞车，可巧设机关布景，即使表演上天入地、神出鬼没的戏，也能运用自如。现存于顶层舞台上的辘轳绞车，是表现升降布景的机关。底层舞台的底部有一眼深水井和五个方形水池，为了增强演唱时的共鸣效果和为水法布景提供真水上台。

 

京剧这种谱式化的艺术形式在宫廷中得到了充分的认可。京剧逐步成为了中国的国剧。清朝是封建社会，君权在人民的思想中有着不可动摇的地位。所以宫廷认可了京剧，那么民间也就必然认可了京剧。封建社会等级制度森严，脸谱的谱式的规定就变得非常严格，被宫廷认可的谱式就作为了当时脸谱的标准谱式，这就加强了脸谱的谱式化。
