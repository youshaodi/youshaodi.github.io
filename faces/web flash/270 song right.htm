宋代英杰

作者：人物的谱式是太监脸，显然这个人物不是太监。使用太监脸，可见人们对侵略者的褒贬。

白天佐 ( 白花三块瓦脸 ) ，《破洪州》一剧中的人物，取材于《杨家府演义》。白天佐是辽国大将。

作者：只有象庞统、包拯这样长相极为特殊的老生才被划入净行，拥有脸谱。

包拯 ( 黑整脸 ) ，《铡美案》一剧中的主角，取材于《秦香莲鼓词》。“鼓词”主要流行于中国北方，是一种又说又唱的成本大书，作品主要写朝廷内忠奸的斗争和抵御外族入侵的战争，虽多是帝王将相的活动，却多少反映了人民的爱憎和愿望。《秦香莲》是一出传统戏曲剧目，许多剧种都有此剧目，故事讲宋王朝龙图阁直学士包拯 (999 一 1062) 不畏权贵，铁面无私，怒铡贪图富贵，残害妻儿，灭绝人性的当朝驸马陈世美的事。

巴若里 ( 黑碎花脸 ) ，《状元媒》一剧中的人物，此剧是一出新编历史剧。巴若里是辽将，在宋王率柴郡主赴边关狩猎时，曾擒住柴郡主。杨延昭在潼台救驾，将柴郡主救回，郡主对杨有情，赠杨珍珠衫，后由状元吕蒙正为媒，使郡主与延昭二人成婚。
崔子建 ( 紫十字门花脸 ) ，《紫金带》一剧中的人物，剧目出处不详。该剧讲崔、杨、佘三家儿女婚配的事。

作者：眉毛变化成了两道弯，使人感觉非常圆滑。

傅龙 ( 老红六分脸 ) ，《状元媒》一剧中的人物，宋王的大臣，因冒认救驾之功受罚。

作者：标准十字门脸。

高旺 ( 黑十字门脸 ) ，《牧虎关》一剧中的人物，出自《杨家府演义》。高旺是杨家部将，遭眨后一家失散，后在牧虎关遇子，一家从此团聚。

哈迷蚩 ( 武丑脸 ) ，《潞安州》一剧中的人物，金兵的军师。《潞安州》、《锤震金蝉子》、《求贤鉴》、《挑滑车》、《飞虎梦》、《栖梧山》、《八大锤》、《牛头山》等剧，均取材于《说岳全传》。此书是清人钱彩在各种《岳传》的基础上加工增订而成的讲史小说，全称《精忠演义说本岳王．全传》。内容以南宋抗金名将岳飞 (1103 一 1142) 及其将士抗金故事为中心，揭露了张邦昌、秦桧等与金贵族内外勾结灭宋的罪行。《说岳全传》在人物描写和情节安排上，细致深刻，性格鲜明，使岳飞不平凡的一生，完整地表现了出来，一些可歌可泣的故事感人极深。

韩昌 ( 红碎花脸 ) ，《杨排风》一剧中的人物，此剧据杨家将故事编成。韩昌是辽国大将。
韩章 ( 蓝三块瓦脸 ) ，《三侠五义》一剧中的人物，此剧为同名长篇小说编成的连六本戏。韩章、卢方、徐庆、蒋平和白玉堂五人结义，号称“五鼠”。至京城与“御猫”展昭比试，大闹东京，后共归开封府。韩章在“五鼠”中行二。小说原名《忠烈侠义传》，清人据单弦艺人石玉昆说唱的《龙图公案》及听书人的笔录本《龙图耳录》改编而成。前半部主要写开封府尹包公在展昭、白玉堂等侠客的帮助下受理民冤、神断疑案，同奸臣庞大师父子斗争的故事；后半部主要记述江湖侠客们除暴安良的行为。故事情节曲折，人物刻画生动。书中反映了封建统治的黑暗残暴，也宣扬了封建伦理道德和因果报应等迷信思想。是传统戏曲剧目的又一重要题材。
何元庆 ( 红花三块瓦脸 ) ，《栖梧山》一剧中的人物，栖梧山守将，与岳飞交战时，被岳数次擒而纵之，使何心服归顺。

作者：额头上的花纹要一气呵成，勾画很有难度。

呼延庆 ( 揉黑碎花脸 ) ，《呼延庆打擂》一剧中的主角，此剧取材于《金鞭呼家将演义》。 136. 王文 ( 白尖三块瓦脸 ) ，《杨门女将》一剧中的人物，这是一部新编历史剧。讲佘太君和穆桂英率杨门女将战败西夏国王王文的事。

呼延赞 ( 黑碎花脸 ) ，《龙虎斗》一剧中的主角之一，剧目出自《杨家府演义》。此书是明人所作的一部宣扬爱国主义的讲史小说。歌颂了抗击辽、西夏的杨家将。生动地描写了北宋名将杨业 ( 杨老令公 ) 及其子孙世代英勇抗击外患的故事。历述了杨延昭 ( 杨六郎 ) 、杨宗保、穆桂英、焦赞、孟良、杨文广等英雄人物，一面坚决抗击辽、西夏的侵犯，一面同朝廷中权奸所作斗争的感人事迹。最后以杨文广之子杨怀玉因宋帝屡信谗言，迫害忠良，愤而于黑夜锄奸，率领全家上太行山作结束。以《杨家府演义》为剧目的戏很多，如《龙虎斗》、《金沙滩》、《五台山》、《穆柯寨》、《破洪州》、《清官册》等。《龙虎斗》是讲北宋大将呼延赞为父报仇，与赵匡胤交战的情节。赵被鞭打坠马，头上忽现金龙，赞大惊亦坠下，头上现出虎形，二人相互倾心，赞即归降。
蒋平 ( 武丑脸 ) ，《铜网阵》一剧中的主角，此剐取材于《三侠五义》，故事讲赵王谋篡，设铜网阵，颜查散奉旨偕白玉堂巡视襄阳，印信被盗，白陷阵被箭射死，蒋平潜入寒潭捞回印信。蒋平在“五鼠”中行四。
焦赞 ( 黑十字门脸 ) ，《穆柯寨》一剧中的人物，北宋元帅杨延昭帐下的大将。与孟良情同手足。

作者：额头的圆象征：禅。

金蝉子 ( 黑金碎花脸 ) ，《锤震金蝉子》一剧中的人物，金兵将领。故事讲金蝉子连败宋将，岳飞挂免战牌以避其锐气，岳飞子岳云不明父意，怒砸免战牌；岳飞欲斩子，大将牛皋劝阻，使岳云戴罪出战，锤击金蝉子得胜而归。

卢方 ( 老红三块瓦脸 ) ，《三侠五义》一剧中的人物，为“五鼠”之首。
马汉 ( 元宝脸 ) ，《铡美案》一剧中的人物，包拯的随身校尉之一。

作者：孟良的招牌法宝火葫芦被画在了脸谱上。可见脸谱就是人物的招牌。

孟良 ( 红十字门葫芦脸 ) ，《穆柯寨》一剧中的人物，取材于《杨家府演义》。孟良是宋军大将，宋辽交战时，辽军设“天门阵”，需“降龙木”方可破阵，孟良与焦赞至穆柯寨强取，被穆桂英打败，杨宗保助战亦被擒。穆桂英爱慕宗保，愿归顺宋营，宗保允婚，穆献出降龙木共破天门阵。

牛皋 ( 黑十字门花睑 ) ，《飞虎梦》一剧中的人物，岳飞的得力战将，在往救藕塘关得胜后，守将金节以小姨戚赛玉许婚，洞房中夫妻比武。
陶洪 ( 老武丑脸 ) ，《打瓜园》一剧中的主角，取材于《风云会》传奇。五代时，油贩郑恩 ( 子明 ) 路经陶洪瓜园，偷摘其瓜，郑理亏而不认错，反与之相斗，陶喜郑骁勇，招郑为婿。

作者：褒贬非常明显。这也是现代戏中，为什么把用外在形象来区分人物的好坏叫做脸谱化。

天庆王 ( 水白整脸 ) ，《金沙滩》一剧中的人物，取材于《杨家府演义》。天庆王是辽国首领。
王朝 ( 紫三块瓦脸 ) ，《铡美案》一剧中的人物，包拯的随身校尉之一。
徐庆 ( 绿碎花脸 ) ，《三侠五义》一剧中的人物，在“五鼠”中行三。
杨延德 ( 白僧脸 ) ，《五台山》一剧中的主角之一，取材于《杨家府演义》。杨六郎延昭至北国搬取其父遗骨，归时经五台山投宿庙中，偶遇其兄五郎延德，互相盘问方相认，兄弟悲喜交集。
杨延嗣 ( 黑碎花脸 ) ，《金沙滩》一剧中的人物，是杨继业第七个儿子，在去金沙滩赴“双龙会”时大战辽将，与五郎、六郎突围回到宋地，后被奸臣潘洪所害。
张天龙 ( 黑碎象形脸 ) ，《双沙河》一剧中的人物，此剧据宋代故事编成。杨家将征土番国，土邦驸马张天龙与公主玉宝、玉珍出战，两位公主慕宋武士二高能、杨仙童貌美，反助高、杨杀张归宋。
赵虎 ( 黑碎花脸 ) ，《铡美案》一剧中的人物，包拯的随身校尉之一。

赵匡胤 ( 红整脸 ) ，《龙虎斗》一剧中的主角之一，宋王朝开国皇帝，公元 960 ― 976 年在位。

郑子明 ( 黑歪花脸 ) ，《斩黄袍》一剧中的人物，剧目出自《飞龙传》。五代时周世宗，柴荣驾崩，大将郑恩 ( 子明 ) 、高怀德拥赵匡胤为帝，改宋。河北韩龙进妹素梅，受封游御街，郑恩怒批其颊，韩逃入宫中与妹同在赵前进谗，赵醉后不顾高怀德等忠谏，怒斩郑。郑妻陶三春闻讯进兵围困京城，赵酒醒自咎，央高为调入，高斩韩龙后进行调解，陶斩赵所服黄袍泄忿，故名“斩黄袍”。
