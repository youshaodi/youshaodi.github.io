大鹏 ( 绿象形脸 ) ，《狮驼岭》一剧中的妖怪，本是如来佛身边的护法之鸟，私自下界作怪，被如来佛降伏收回。

地煞 ( 黑神仙脸 ) ，《安天会》一剧中天宫的星宿。

二郎神 ( 金三块瓦神仙脸 ) ，《闹天宫》一剧中奉命捉拿孙悟空的天神。他是玉帝的外甥，带哮天犬，亦善变化，是孙悟空的劲故。
龟帅 ( 小妖脸 ) ，《水帘洞》一剧中龙宫的水族。
黄袍怪 ( 黄象形脸 ) ，《宝象国》一剧中的妖怪，为“二十八宿”中奎木狼私自下界。孙悟空三打白骨精，被唐僧逐回花果山。唐僧在前进途中被黄袍怪所擒，幸黄袍怪之妻为宝象国公主，将唐僧释放。猪八戒智激悟空前来，降伏黄袍怪，将宝象国公主送回。
金钱豹 ( 金象形脸 ) ，《红梅山》一剧中的妖怪。金钱豹占据红梅山，欲强娶乡绅邓洪之女，唐僧师徒借宿知其事，孙悟空幻化降妖，金钱豹又摆阵困悟空，悟空请天兵将其降伏。

作者：巨灵神的脸谱分成上下两个部分，上面是婴儿的脸，慈眉善目。下面则青面獠牙。巨灵神登场时先把头低着，用手遮住青面獠牙的真脸，只露出孩童的脸。当走到戏台正中央时再突然露出恐怖的真面目。非常有舞台效果。

巨灵神 ( 黑花元宝脸 ) ，《闹天空》一剧中李天王的先锋官。

亢金龙 ( 黑象形脸 ) ，《琵琶洞》一剧中的神仙。琵琶洞蝎子精将唐僧捉去逼婚，孙悟空、猪八戒与之格斗，均被蝥伤。孙悟空请昂日鸡、亢金龙将妖降伏，救出唐僧。亢金龙为“二十八宿”之一。《琵琶洞》、《弼马温》、《闹天宫》、《双心斗》、《安天会》、《石猴出世》、《芭蕉扇》、《高老庄》、《流沙河》、《宝象国》、《狮驼岭》、《无底洞》、《红梅山》、《水帘洞》等剧，均取材于中国古典长篇小说《西游记》。此书是明吴承恩 (15107 一 15827) 在民间流传的唐僧取经故事和有关话本、杂剧的基础上，经再创作而成长篇小说。叙述美猴王孙悟空出世，通过大闹天宫，转而为保护唐僧去西天取经，沿途降妖伏魔，用幻想的形式反映社会矛盾，歌颂了孙悟空不畏强暴、战胜困难的顽强精神。除塑造了孙悟空、唐僧、猪八戒等几个主要形象外，还用大笔勾勒了一批神魔鬼怪的形象。作者运用浪漫主义的创作方法，以丰富的想象，曲折的情节，诙谐的语言写成一部规模宏大，结构完整的巨著，成为中国戏曲取之不尽的剧目宝库。
孔宣 ( 绿象形脸 ) ，《百草山》一剧中的神仙，此剧取材于《钵中莲》传奇。百草山旱魃化身为王大娘，取死人食罐炼成磁缸施法，观音遣土地幻作补锅匠碎其缸，破其妖法。大鹏、孔宣、通雀鸟等下界共除旱魃。
雷公 ( 蓝象形鸟脸 ) ，即雷神，《闹天宫》一剧中天将之一，善用五雷轰击。
李天王 ( 红尖三块瓦神仙脸 ) ，名李靖，《闹天宫》一剧中奉玉帝命，率天兵擒拿孙悟空的天神，号称托塔李天王。
刘天君 ( 红三块瓦脸 ) ，《闹天宫》一剧中天宫四大天王之一。
罗T ( 瓦灰象形睑 ) ，《闹天宫》一剧中天宫的星宿，天将之一。
猫神 ( 白象形脸 ) ，《无底洞》一剧中的神仙。唐僧被白鼠精摄去逼婚，孙悟空救师，盗取神牌，请天兵捉妖俱不胜，又请猫神至，才降服白鼠精。
马天君 ( 白花三块瓦脸 ) ，即马王爷，《闹天宫》一剧中天宫四大天王之一。

牛魔王 ( 金象形脸 ) ，《芭蕉扇》一剧中的主角之一，铁扇公主的丈夫，原是孙悟空的好友。

青龙 ( 蓝象形脸 ) ，《闹天宫》一剧中天宫的星宿，为东方武神
青狮 ( 绿象形脸 ) ，《狮驼岭》一剧中的妖怪。唐僧师徒四人至狮驼岭，遇青狮、白象，大鹏阻路，孙悟空虽运用变化破阴阳瓶，力降狮、象，却中大鹏之计，唐僧被擒。悟空请来如来佛才降伏三妖。
齐天大圣 ( 红象形脸 ) ，《闹天宫》一剧中的主角。孙悟空受封为“齐天大圣”后，在蟠桃园看守桃园。一日，从摘桃仙女处得知他没有被邀参加蟠桃盛会，就自行闯进瑶池，吃了蟠桃会上的仙桃仙洒，又入兜率宫，吞了太上老君的金丹，最后闯出南天门，并打败追捕他的天兵天将。

作者：说实话，很难看。表现的是红日，金佛，星空，宇宙。但综合效果很糟糕。

如来 ( 金神仙脸 ) ，《双心斗》一剧中的佛祖。唐僧逐走孙悟空后，六耳猕猴假冒孙悟空，观音令悟空往降，天宫、地府都辨不清真假。如来佛指出假悟空来历，孙悟空才将其打死。

煞神 ( 黑花元宝脸 ) ，《琼林宴》一剧中的神怪，此剧取材于《三侠五义》。讲书生范仲禹的妻儿，被告老还乡的太师葛登云抢走，范至葛家索妻，葛将其灌醉，密遣家丁葛虎刺之，葛虎反为煞神所杀。
沙悟净 ( 蓝僧脸 ) 是《流沙河》剧中的水怪。此剧讲唐僧师徒三人行至流沙河，孙悟空请观音菩萨收水怪为徒，赐名沙悟净，共保唐僧过河的故事。
通臂猿 ( 红象形脸 ) ，《石猴出世》一剧中的老猴，他建议猴王孙悟空去学道。孙悟空至昆仑山拜师，终于学成七十二变筋斗云，回山劈死抢战花果山的混世魔王。
通雀鸟 ( 红象形脸 ) ，《百草山》一剧中的神鸟。

玉帝 ( 银三块瓦神仙脸 ) ，《弼马温》一剧中天宫的玉皇大帝，因怕孙悟空神通广大，以天上官爵加以羁縻，封为弼马温。悟空知其底细后，大闹御马监，反下天宫。

赵天君 ( 黑六分花脸 ) 是武财神，《闹天宫》一剧中天宫四大天王之一。
猪八戒 ( 揉黑象形脸 ) ，《高老庄》一剧中的主角。在此剧中，猪八戒幻化人形，占娶高员外之女，孙悟空保唐僧取经路过高老庄，悟空将其降伏，同往西天取经。
