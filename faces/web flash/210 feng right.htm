封神演义

陈奇 ( 红花三块瓦脸 ) ，《青龙关》一剧中的人物，是“青龙关”殷守将丘引的督粮官，善左道。郑伦、陈奇即“哼、哈”二将。

作者：在京剧脸谱中，紫色的概念比较模糊，深红色，紫色和黑色有的时候可以混用。

崇黑虎 ( 黑碎花脸 ) ，《进妲已》――剧中的人物，崇侯虎的弟弟，官拜曹州侯。

崇侯虎 ( 白花三块瓦睑 ) ，《进妲已》中的人物，殷纣王四镇诸侯之一，取材于《封神演义》。

作者：雷震子的脸谱非常有味道。基色蓝色既说明了雷震子性格的桀骜，就恰好表明雷震子在蓝天中翱翔。雷震子下巴上的图案臣为鸟形图案。在所有有翅膀的角色上，都会有这个图案。

雷震子 ( 蓝象形脸 ) ，《百子图》一剧中的人物，出自《封神演义》。雷震子是西伯侯姬昌 ( 即周文王 ) 奉诏去京城朝歌朝见纣王，途经燕山遇雨时所得的第一百个儿子，故名《百子图》。

土行孙 ( 黄象形睑 ) ，《三山关》一剧中善土遁仙法的战将。取材才于《封神演义》

作者：刘芬连通常用连表现年老的英雄。脸两侧的花纹刚好像是两撇下垂的白眉毛。额头上的第三志愿竟是稳态势的招牌。

闻仲 ( 红六分脸 ) ，《大回朝》一剧中的主角，殷朝的太师，取材于《封神演义》。

郑伦 ( 绿尖三块瓦脸 ) ，《青龙关》一剧中的人物，此剧出自中国古典长篇小说《封神演义》。这是一部成书于明代隆庆、万历年间 (1567 ― 1620) 的神话小说。所写的殷 ( 商 ) 周争斗的故事，不同于史实，其人物和情节大都是幻想出来的，并加上了神话色彩，利用神话中的“神仙”、“道法”、“法宝”之类的新奇幻想，吸引读者的兴趣。内容始自商末 ( 约公元前 ll ―前 9 世纪 ) 政治纷乱和武王伐商，末以姜子牙封诸神及周武王分封诸侯作结。“青龙关”是殷周争夺的一座城关。郑伦是周文王的部将。
