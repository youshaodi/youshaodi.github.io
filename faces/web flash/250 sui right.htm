隋唐演义

安殿保 ( 白尖三块瓦睑 ) ，《独木桥》一剧中的人物，取材于《征东全传》故事。安殿保是独木关番军守将，在连败唐将，又将张士贵之子及婿擒去后，累冒薛仁贵战功的张士贵不得已，至薛病榻前求告；薛扶病出战，枪挑安殿保。
巴杰 ( 黑碎花脸 ) ，《巴骆和》一剧中的人物。巴杰是酸枣岭巴家寨主。

巴兰 ( 红三块瓦老脸 ) ，《百花公主》一剧中的人物，《百花公主》是一出新编历史剧。唐安西国百花公主中了奸细海俊之计，不听老将巴兰劝阻，孤军作战被困。巴兰领兵解围，救出百花公主，并处死奸细。

鲍自安 ( 白老三块瓦脸 ) ，《巴骆和》一剧中的人物，剧目出自《绿牡丹》。这是一部由清人所作的长篇小说，又名《四望亭》、《龙潭鲍骆奇书》。故事以骆宏勋和花碧莲的婚姻为线索，描写一群江湖侠客的故事，充满了忠孝节义的封建说教。《巴骆和》一剧讲公子骆宏勋赴山东成亲，路过酸枣岭，因言语不和，与巴家寨主巴杰造成误会，在鲍自安及和尚黄胖等江湖英雄的调解下，巴、骆两家和好。

作者：绿（lu）林好汉，绿(lv)林好汉

程咬金 ( 绿碎花脸 ) ，《贾家楼》一剧中的人物，《说唐》故事之一。单雄信前去为秦琼母亲贺寿，广邀江湖好汉前往祝贺，并在贾家楼结拜。这时，程咬金因再劫皇杠被捕，徐勖分遣结拜兄弟劫牢反出山东。

窦一虎 ( 绿碎花脸 ) ，《棋盘山》一剧中的人物，取材于《征西全传》。薛丁山与母及妹金莲同往锁阳关，途经棋盘山，寨主窦一虎与妹仙童擒住薛丁山，程咬金劝窦降唐，一虎欲娶薛丁山之妹金莲为妻，程佯允，在暗中反促仙童与丁山完婚。

作者：在眉心画上人物的武器是京剧脸谱中常用的区分人物的方法。

盖苏文 ( 蓝花三块瓦脸 ) ，《淤泥河》一剧中的人物，取材于《征东全传》故事。盖苏文是辽将，唐太宗出猎，遇盖苏文大惊而逃马蹄陷入淤泥河中，盖苏文逼写降表，薛仁贵突至救驾，力败盖苏文。

郭子仪 ( 老红六分脸 ) ，《满床笏》一剧中的主角之一，取材于《隋唐演义》。《隋唐演义》是清人褚人获作的讲史小说。作者据《隋唐志传》、《隋炀帝艳史》以及其他民间传说改写而成。从隋文帝伐陈，杨广阴谋夺取太子位开始，到唐明皇之死为止，共写了 170 多年间的历史故事。全书分三段：一是秦琼、单雄信等英雄人物的聚散；二是隋炀帝和朱贵儿的关系；三是唐明皇与杨贵妃的结合。京剧根据小说的故事，编演了一出出生动有趣的剧目。《满床笏》讲汾阳王郭子仪六十寿辰，子婿纷至、笏堆满床，幼子暖妻升平公主持贵不至，暖气而欧之。公主向父母哭诉，郭子仪绑子上殿请罪，唐王不加罪，反晋暖官阶，劝暖和公主夫妻和睦。
贺天龙 ( 白尖三块瓦脸 ) ，《雁荡山》一剧中的人物，据隋唐故事新编的历史剧。唐末起义军首领孟海公率部追歼隋雁荡山守将贺天龙，经山地战、水战、攻城战，终于获胜，全歼隋军。

作者：额头上的戒疤是僧脸的标志。

黄胖 ( 黄僧脸 ) ，《巴骆和》一剧中的人物。黄胖是位武艺高强的和尚。

作者：象形脸，一目了然。

胡理 ( 武丑象形脸 ) ，《巴骆和》一剧中的人物，是个店掌柜，调解巴杰、骆宏勋两家不和的成员之一。

作者：额头上的花纹非常有气势。

金甲 ( 黄碎花脸 ) ，《贾家楼》一剧中的人物，单雄信等 36 位结义兄弟之一。

作者：人物额头上有鹰爪的痕迹，又来区分李克用和黄盖。

李克用 ( 红六分脸 ) ，《沙陀国》一剧中的人物，出自《残唐五代史演义》。唐末，黄巢起义，唐僖宗遣程敬思至沙陀国李克用处搬兵。李克用是沙陀国首领，被封晋王。

李密 ( 紫六分睑脸 ) ，《断密涧》一剧中的人物，取材于《说唐》故事。李密 (582 ― 618) 是瓦岗寨众英雄之首，瓦岗寨众将归唐后，李密被迫与王伯党相随降唐，后又反唐，被唐王李世民追至断密涧，因劝降不从，而被射死。

作者：脸谱的下半部分是鸟翅膀的象形图案。

李元霸 ( 黑象形鸟睑 ) ，《四平山》一剧中的人物，唐王李渊之子，勇猛无敌，为隋唐第一条好汉。《四平山》一剧取材于《说唐》故事，讲隋炀帝被阻四平山，急召李元霸往救的事。

孟觉海 ( 红碎花脸 ) ，《雅观楼》一剧中的主角之一，唐末起义军领袖黄巢的部将。
秦英 ( 红象形鸟脸 ) ，《金水桥》一剧中的人物，传统剧目之一，出处不详。秦怀玉出征，其子秦英在金水桥前将作恶多端的詹国丈打死，银屏公主绑子上殿，唐太宗欲斩，长孙皇后讲情，令公主赔礼，秦英出征救父赎罪。

单雄信 ( 蓝碎花脸 ) ，《锁五龙》一剧中的人物，取材于《说唐》。《说唐》，全称《说唐演义全传》，清人所作讲史小说。它从隋文帝 ( 公元 58l 一 604 年在位 ) 平陈 ( 公元 589 年 ) ，统一南北朝说起，一直写到唐王李世民削平群雄，登上宝座，建立唐王朝为止。书中塑造了不少英雄好汉，如单雄信、程咬金、尉迟恭等，多少年来一直为群众所喜乐见；并以瓦岗寨好汉的聚散为中心，再现了十八路反王、六十四路烟尘的动乱时代。《锁五龙》一剧讲唐初，郑王王世充部将单雄信被唐将尉迟恭所擒，唐秦王李世民苦苦劝降，单不从，乃绑赴法场。临刑前，李世民与已投唐的单雄信的结义兄弟徐勤、罗成、程咬金等来生祭，单痛斥诸人，决心赴死。

苏宝童 ( 黑碎花脸 ) ，《界牌关》一剧中的人物，取材于《征西演义》故事。界牌关是个要隘，苏宝童是界牌关守将。

童环 ( 红碎花脸 ) ，《贾家楼》一剧中的人物，单雄信等 36 位结义兄弟之一。

作者：绿林好汉，额头上有癞蛤蟆的图案。可见此人的长相。

王彦章 ( 绿碎花象形脸 ) ，《五龙斗》一剧中的人物，取材于《残唐五代史演义》。草莽英雄王彦章，在狗家疃力战李克用、李嗣源、石敬塘和刘智远等不敌，自刎而死。

猩猩胆 ( 蓝碎象形脸 ) ，《摩天岭》一剧中的人物，《征东全传》中的故事。猩猩胆是摩天岭的守将，自恃山路险要，防卫不力，被唐大将薛仁贵 (614 ― 683) 施计，大破摩天岭。
辛文礼 ( 黑碎花脸 ) ，《虹霓关》一剧中的人物，传统剧目之一，出处不详。辛文礼是虹霓关守将，被王伯党射死。
薛刚 ( 黑碎花脸 ) ，《九锡宫》一剧中的人物，取材于《薛刚反唐》故事。奸臣张天左、张天右私操禁兵，却向唐王进谗，加害薛丁山、薛刚父子。程咬金封王后，住九锡宫，闻讯后上殿怒斥二张，唐王知情后赦免―了薛氏父子。
薛葵 ( 黑碎花睑脸 ) ，《徐策跑城》一剧中的人物，《薛刚反唐》故事之一。薛蛟被诬含冤，持徐策书信至韩山，与薛刚、薛葵父子合兵攻长安。徐策闻报不顾年迈，亲上城楼观望。又飞跑入朝为薛蛟雪冤，迫胁朝廷斩了奸臣张泰，并劝薛刚、薛葵收兵。
杨林 ( 老红六分睑 ) ，《打登州》一剧中的人物，取材于《倒铜旗》传奇。杨林是隋朝靠山王，得知秦琼 (? 一 638) 因皇杠被劫，被捕解至登州，瓦岗众英雄将乔妆混入登州营救秦琼的消息。杨林设计欲歼之，反被王伯党射灭红灯，乘乱救出。

作者：最标准的紫脸，推荐记忆。

尉迟恭 ( 黑六分脸 ) ，《白良关》一剧中的人物。尉迟迟恭 (585 ― 658) 是唐初大将，在讨伐白良关时，阵前以钢鞭为证，与其失散之子尉迟宝林相会，共破白良关，杀死仇人刘闺帧。此剧因是尉迟恭父子在白良关相会，故又名《父子会》。

宇文成都 ( 黄花三块瓦脸 ) ，《南阳关》一剧中的人物，出自《说唐》故事。宇文成都是杨广 ( 即隋炀帝 ) 的部将，为隋唐第二条好汉。

周德威 ( 红三块瓦脸 ) ，《珠帘寨》一剧中的人物，取材于《残唐五代史演义》。周德威是珠帘寨寨主，李克用起兵至珠帘寨时，周阻路比箭，李一箭射双雕，使周心服归降。
朱温 ( 绿花三块瓦脸 ) ，《雅观楼》一剧中的人物，剧目出自《残唐五代史演义》。此书又名《五代残唐》，系根据《唐书》、《五代史》和有关的讲史小说编成。故事起于黄巢起义 ( 公元 875 年 ) ，终于赵匡胤胨桥兵变 ( 公元 960 年 ) 。《雅观楼》一剧讲晋王李克用与梁王朱温会宴于雅观楼，起义军部将孟觉海兵至，朱与李以玉带为赌，晋王部将十三太保李存孝生擒孟觉海，怒夺玉带，使朱、孪生嫌。
